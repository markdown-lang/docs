## 说明

1. 一个 git 仓库可包含一个或多个子模块，本编码描述的是项目中存的内容种类
2. `项目管理` 主要存储 Excel、Word 或图片等原始业务文档或管理文档等
3. `设计文档` 存储数据库表结构、界面原型、API 等设计文档，用 Markdown 编写
4. `21(前端业务代码)` 指
   1. 能同时构建 `31(浏览器端业务代码)` 和 `41(移动端业务代码)`
   2. 能同时构建 `51(桌面端业务代码)` 和 `41(移动端业务代码)`
