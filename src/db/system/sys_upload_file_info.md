# `sys_upload_file_info` - 上传文件信息表

集中管理上传文件的基本信息。

## 字段

| 字段名      | 注释     | 类型         | 默认值 | 可空 | 单位 |
| ----------- | -------- | ------------ | ------ | ---- | ---- |
| dbid        | 主键     | bigint(20)   |        | 否   |      |
| name        | 文件名称 | varchar(64)  |        | 否   |      |
| path        | 文件路径 | varchar(512) |        | 否   |      |
| create_by   | 创建人   | bigint(20)   |        | 否   |      |
| create_time | 创建时间 | datetime     |        | 否   |      |

## 约束

1. 主键: `pk_sys_upload_file_info`
2. 外键: 无
3. 唯一：无

## 索引

无

## 说明

1. `name(文件名称)` 存的是文件的原始名字
2. `path(文件路径)` 存是相对路径，其中的文件名称是重新命名过的
